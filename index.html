PANOPTES

An open-source, inventory-driven vulnerability management platform. Panoptes correlates real software state with real CVEs — eliminating blind spots, false negatives, and scanner guesswork.

GitHub Architecture

Design Philosophy

Traditional scanners infer risk from exposed services. Panoptes takes a different approach: it observes the system itself, builds a complete inventory, and applies deterministic logic to vulnerability intelligence.

Inventory First

Endpoint sensors enumerate installed software, versions, patches, and configurations — even if the software is dormant or unreachable from the network.

Logic-Based CVE Matching

Instead of one signature per vulnerability, Panoptes evaluates affected version ranges, vendor logic, and real-world applicability.

Explainable Results

Every finding is backed by evidence: software path, version, detection logic, and reference data — no black boxes.

Architecture

Spectra — Sensor

A native endpoint sensor responsible for collecting precise inventory data directly from the operating system.

Iris — Correlation Engine

A backend analysis engine that normalizes inventory data and correlates it with vulnerability intelligence sources.

Future UI & API

Planned interfaces for visualization, automation, and integration with SOC and vulnerability management workflows.
