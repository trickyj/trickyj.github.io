Panoptes

See Everything. Miss Nothing. Panoptes is an open‑source vulnerability management platform designed to eliminate blind spots by combining deep asset inventory with intelligent CVE correlation.

View on GitHub Architecture

Why Panoptes?

Inventory‑Driven

Collects precise software, version, and configuration data directly from endpoints instead of relying only on network signatures.

Fewer False Negatives

Detects vulnerable software even if it is not actively listening on a port or missed by traditional scanners.

Open & Transparent

Fully open‑source, allowing security teams and researchers to understand, verify, and extend detection logic.

Architecture

Spectra (Sensor)

A lightweight endpoint sensor that runs on systems to collect detailed inventory information such as installed software, versions, patches, and OS details.

Iris (Correlation Engine)

A backend analysis engine that correlates collected inventory data with vulnerability intelligence sources to identify real exposures.

Future UI & API

Planned dashboards and APIs to visualize risk, track remediation, and integrate Panoptes into SOC and VM workflows.

Use Cases

Blue Teams

Gain accurate visibility into vulnerabilities and validate scanner findings before remediation.

Security Researchers

Study how vulnerabilities map to real‑world software inventory and experiment with detection logic.

Compliance & Audit

Provide defensible evidence of vulnerability presence and remediation based on factual system data.
